# Deployment Baseline

## 目标

记录 `{{PROJECT_NAME}}` 的长期部署与运行基线，避免环境约束只留在聊天记录中。

## 待补充项目

- 运行环境：操作系统、CPU、内存、磁盘约束
- 环境管理：包管理器、虚拟环境或容器约定
- 域名与路由：主站、API、后台、静态资源的分流方式
- 数据依赖：数据库、缓存、对象存储、消息队列
- 配置与密钥：环境变量、密钥来源、非生产演示账号策略
- 监控与回归：健康检查、smoke test、日志与告警入口

## 使用规则

- 只记录会长期影响部署和运行的稳定约束
- 暂时性的调试命令或一次性操作不要放在这里
- 当部署拓扑、域名结构或资源配额发生变化时，同步更新本文件
